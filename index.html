_______ React Apps _______

APIFLIX is a React app I made using TMDB.org api and using Google Firebase. It is a clone of Netflix. Each category maps through the api to show the results for that search result. All the critical data is stored on the api and not the server.

Expense It! is a front end React app I made that utilizes voice recognition, local storage, context api, and custom hooks. It is an expense tracker that has a very robust UI for desktop and mobile. User can speak or input the fields to add an income or expense to the page. User can delete transaction with voice or input as well.

Kryptokit is a React app I made with a team. The user must log in through authenicating and passing validation; they can then see coingecko api listed cryptocurrencies live. From there they can buy, sell or trade currencies in this mock app. It displays all data realtime via Coingecko API.

SimplyNStock is a React app to manage inventory across multiple locations. It uses MongoDB and Node.js. Users may add an item to the inventory by selecting the physical locaiton as well as the warehouse it would be located at.

Things to Do is a React app using useEffect and useState to allow user to create a list that maintains through reload. Once user clicks on the task as it is completed it will be strikethrough. User can then remove it by clicking the "X" next to the item!

Prattle Chat App is a React app I made using Google and Facebook Auth/IO. It does many api calls using Chat Engine as the display. User can log in with their Google or Facebook credentials via Firebase to join a customized chat program.
