Projects

SimplyNStock is an app to manage inventory across multiple locations. User may add an item to the inventory by selecting the physical locaiton as well as the warehouse it would be located at. You can also add a photo link and a description about the item. The main dashboard page displays items that are considered low stock (less than 10), the warehouse locations for quick filtering, and the last three added items into the database.

KryptoKit is an app that is used to manage a wallet for buying and selling crypto currency. This app pulls from coinbase api to have current prices on all currencies. You are able to make users as well.
