Portfolio

This is just a few of the projects I have done since graduating Coding Dojo Bootcamp.

Expense It! is a front end React app I made that utilizes voice recognition, local storage, context api, and custom hooks. It is an expense tracker that has a very robust UI for desktop and mobile. User can speak or input the fields to add an income or expense to the page. User can delete transaction with voice or input as well. A graphical chart to represent the data being entered also shows up.

Things to Do is a React app using useEffect and useState to allow user to create a list that maintains through reload. Once user clicks on the task as it is completed it will be strikethrough. User can then remove it by clicking the "X" next to the item!

APIFLIX is a React app I made using TMDB.org api and using Google Firebase. It is a clone of Netflix. User can hover over each thumbnail of the movie which pulls from multiple api locations for Netflix like effects using transform. Each category maps through the api to show the results for that search result. You can click on the thumbnail for a trailer for each movie/tv show. All the critical data is stored on the api and not the server.

SimplyNStock is a React app to manage inventory across multiple locations. It uses MongoDB and Node.js. Users may add an item to the inventory by selecting the physical locaiton as well as the warehouse it would be located at. You can also add a photo link and a description about the item. The main dashboard page displays items that are considered low stock (less than 10), the warehouse locations for quick filtering, and the last three added items into the database.

Snake is a classic game I made using python to demonstrate more of what object oriented programming looks like. I wanted to see how pygame worked and it took me just a while to make.
